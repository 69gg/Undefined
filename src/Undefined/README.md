# Undefined 主包

本目录包含机器人核心实现与运行时模块。

核心模块：
- `ai/`：模型请求、提示词构建、工具调度、多模态与总结能力
- `arxiv/`：arXiv 论文解析、元信息获取、PDF 下载与发送（自动触发 + AI 工具）
- `bilibili/`：B 站视频解析、下载与发送（自动触发 + AI 工具）
- `memes/`：全局表情包库、多模态解析管线与异步存储
- `services/`：安全、命令、队列与协调服务
- `skills/`：工具与智能体插件系统
- `utils/`：通用工具与跨平台支持
- `webui/`：WebUI 服务端

入口文件：
- `main.py`：程序启动入口
