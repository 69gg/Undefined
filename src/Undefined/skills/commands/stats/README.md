# /stats 命令说明

## 功能
统计最近一段时间的 Token 使用情况，并生成图表和摘要；可选启用 AI 智能分析。

## 用法
- `/stats`
- `/stats <时间范围>`
- `/stats --ai`
- `/stats <时间范围> --ai`

## 参数
- `<时间范围>` 可选，支持：
- `Nd`：最近 N 天，例如 `7d`、`30d`
- `Nw`：最近 N 周，例如 `2w`
- `Nm`：最近 N 个月（按 30 天换算），例如 `1m`
- `N`：直接按天数，例如 `14`
- `--ai` 或 `-a` 可选，显式启用 AI 智能分析（默认关闭）

## 示例
- `/stats`
- `/stats 30d`
- `/stats 2w`
- `/stats --ai`
- `/stats 30d --ai`

## 说明
- 天数最小 1 天，最大 365 天，超出会自动收敛到允许范围。
- 参数解析失败时会回退到默认值（7 天）。
- 默认只发送图表和摘要，不会触发 AI 分析。
