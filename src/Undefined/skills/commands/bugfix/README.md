# /bugfix 命令说明

## 功能
提取指定用户在时间区间内的聊天记录，交给 AI 分析并生成修复报告。

## 用法
- `/bugfix <QQ号1> [QQ号2 ...] <开始时间> <结束时间>`

## 参数
- `<QQ号1> [QQ号2 ...]`：一个或多个目标 QQ 号（纯数字）。
- `<开始时间>`：格式 `YYYY/MM/DD/HH:MM`。
- `<结束时间>`：同上，或 `now`。

## 示例
- `/bugfix 123456 2024/12/01/09:00 now`
- `/bugfix 111111 222222 2024/12/01/09:00 2024/12/01/18:00`

## 说明
- 至少需要 3 个参数（用户列表 + 起止时间）。
- 时间格式或 QQ 号格式不合法会直接返回错误提示。
