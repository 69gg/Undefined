# 社交媒体助手

## 能力

- **Bilibili 搜索**：搜索 B 站视频、番剧、用户信息
- **Bilibili 用户信息**：获取 B 站用户的详细信息
- **全网音乐搜索**：搜索全球范围内的音乐
- **音乐信息获取**：获取音乐的详细信息
- **音乐歌词获取**：获取歌曲的歌词
- **视频随机推荐**：获取随机视频推荐

## 使用方式

### Bilibili 搜索
- 提供搜索关键词（视频名称、UP主、番剧名等）
- 可选指定返回数量（默认为 5）

### Bilibili 用户信息
- 提供目标用户的 B 站 UID

### 音乐搜索
- 提供歌曲名称或关键词
- 可选指定返回结果的序号

### 音乐信息
- 提供音乐名称获取详细信息

### 音乐歌词
- 提供音乐名称和可选的序号获取歌词

### 视频推荐
- 直接获取随机视频推荐

## 返回格式

- 搜索结果包含标题、作者、链接、播放量等
- 音乐信息包含歌手、专辑、时长等
- 歌词以纯文本格式返回
