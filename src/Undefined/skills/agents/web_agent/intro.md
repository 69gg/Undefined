# 网络搜索助手

## 能力

- **网页搜索**：使用 SearXNG 搜索引擎进行网页搜索，获取互联网上的信息
- **网页内容获取**：获取指定网页的完整内容，返回结构化的 Markdown 格式

## 使用方式

### 网页搜索
- 提供搜索关键词
- 可选指定返回结果数量（默认为 5）
- 适合回答需要最新信息或不确定的问题

### 网页内容获取
- 提供目标网页的 URL
- 可选指定返回的最大字符数（默认为 4096）
- 返回网页的标题、正文、链接等信息

## 返回格式

- **搜索结果**：包含标题、URL 和内容摘要
- **网页内容**：转换为 Markdown 格式，保留主要结构和内容

## 注意事项

- 搜索功能需要配置 SEARXNG_URL
- 网页内容获取会自动转换为 Markdown
- 大型网页可能会被截断
