# NagaAgent代码分析助手

该助手提供了针对NagaAgent的代码读取和分析能力，仅当涉及NagaAgent时调用。

## 能力

- **读取文件内容**：读取指定文件的内容，路径相对于工作目录
- **列出目录结构**：列出指定目录下的文件和子目录
- **搜索文件内容**：在文件中搜索指定内容，支持正则表达式
- **按模式查找文件**：使用 glob 模式查找匹配的文件（如 `**/*.py`）
- **读取 NagaAgent 项目介绍**：获取 NagaAgent 项目的详细文档

## 使用方式

### 读取文件
- 提供文件路径，路径是相对于工作目录的
- 文件内容会限制最大长度（10000字符），超长会自动截断
- 自动检测文件编码，支持多种编码格式

### 列出目录
- 提供目录路径，默认为工作目录根目录
- 返回目录下的文件和子目录列表

### 搜索内容
- 提供搜索模式（支持正则表达式）
- 可选指定搜索路径和文件过滤模式
- 默认返回前 50 行结果

### 文件匹配
- 提供 glob 模式，如 `**/*.py`、`code/**/*.py`
- 默认在 NagaAgent 子目录下搜索
- 最多返回 100 个匹配结果

## 返回格式

- 文件内容会显示编码信息
- 搜索结果包含文件路径和匹配行
- 文件匹配返回相对路径列表
