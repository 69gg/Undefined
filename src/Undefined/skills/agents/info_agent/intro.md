# 信息查询助手

## 能力

### 生活信息
- **天气查询**：查询指定城市的天气状况
- **黄金价格**：查询黄金实时价格

### 热搜榜单
- **百度热搜**：获取百度热搜榜单
- **微博热搜**：获取微博热搜榜单
- **抖音热搜**：获取抖音热搜榜单
- **腾讯新闻**：获取腾讯新闻资讯

### 文化娱乐
- **历史今天**：查询历史上的今天发生的事件
- **文昌帝君**：查询升学考试相关资讯

### 网络工具
- **QQ 等级查询**：查询指定 QQ 号的等级信息
- **网络诊断**：tcping（TCP 延迟检测）、speed（网速测试）、net_check（网络检查）
- **Base64 编码/解码**：对文本进行 Base64 编码或解码
- **哈希计算**：计算文本的哈希值（MD5、SHA1 等）
- **Whois 查询**：查询域名的注册信息

## 使用方式

### 天气查询
- 提供城市名称（如"北京"、"上海"）

### 黄金价格
- 直接查询当前金价

### 热搜/新闻
- 直接查询当前热搜/新闻列表

### 历史今天
- 直接查询今天的历史事件

### 文昌帝君
- 直接查询升学相关资讯

### QQ 等级
- 提供目标 QQ 号

### 网络诊断
- 提供目标地址（域名或 IP）
- tcping：检测 TCP 延迟
- speed：测试下载速度
- net_check：检查网络连通性

### 编码工具
- Base64：提供要编码/解码的文本
- 哈希：提供要计算的内容

### Whois 查询
- 提供域名（如：example.com）

## 返回格式

- 天气：温度、湿度、天气状况等
- 热搜：排名、标题、热度
- 新闻：标题、来源、时间
- 历史：事件描述、年代
- 网络诊断：延迟、丢包率、连接状态
- 编码工具：处理后的结果
