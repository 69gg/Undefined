# 📄 文件分析助手

**file_analysis_agent** 是一个强大的文件解析和分析工具，能够处理多种文件格式，帮助你快速理解文件内容。

## ✨ 核心功能

- **多格式支持**：支持文档、代码、压缩包、图片、音频、视频等多种文件类型
- **智能检测**：自动识别文件类型并选择合适的解析方式
- **内容提取**：从 PDF、Word、PPT、Excel 等文档中提取结构化内容
- **代码分析**：支持 Python、JavaScript、Java、C++ 等主流编程语言的代码分析
- **多媒体处理**：支持图片识别、音视频分析
- **灵活输入**：支持 URL 链接和 QQ 文件 ID 两种方式

## 🎯 使用场景

- 📊 **数据提取**：从 Excel、PDF 中批量提取数据和表格
- 📝 **文档理解**：快速总结 Word、PPT 等文档内容
- 🔍 **代码审查**：分析代码结构、统计行数、检查质量
- 📦 **压缩包检查**：查看或解压 ZIP、RAR、7z 等压缩文件
- 🖼️ **图片识别**：识别图片中的文字、对象和场景
- 🎬 **媒体分析**：处理音频和视频文件

## 📋 支持的文件类型

| 分类 | 支持的格式 |
|------|----------|
| **文档** | PDF, DOCX, PPTX, XLSX, ODP, ODT |
| **代码** | Python, JavaScript, TypeScript, Java, C/C++, Go, Rust, PHP, Ruby, SQL, R |
| **文本** | TXT, MD, JSON, YAML, XML, LOG, CSV, RST |
| **压缩** | ZIP, TAR, GZ, 7Z, RAR |
| **媒体** | PNG, JPG, GIF, MP3, MP4, WEBM |

## 💡 典型用法

```
提取这个 PDF 中的所有数据表格
统计这个代码文件的函数个数
这个压缩包里有什么文件？
分析这张图片的内容
总结这个 Word 文档的要点
```
